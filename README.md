# Personal Expense Tracker

A modern, responsive, and interactive Personal Expense Tracker web application built using HTML, CSS, and JavaScript.  
It helps users manage income and expenses efficiently with real-time calculations and data visualization.

---

## Live Demo

👉 https://starky112.github.io/Personal-Expense-Tracker/

---

##  Features

- ➕ Add income and expense transactions
- 🗑️ Delete transactions instantly
- 📊 Real-time balance calculation
- 📂 Category-based expense tracking
- 📉 Interactive doughnut chart (Chart.js)
- 🌙 Dark / Light mode toggle
- 💾 Local storage support (data persists after refresh)
- 📱 Fully responsive design for all devices
